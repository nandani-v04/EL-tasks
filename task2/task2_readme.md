# Exploratory Data Analysis (EDA) - Titanic Dataset

## 📌 Objective

To understand the dataset using descriptive statistics and visualizations.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 🔍 Steps Performed

1. Loaded dataset using Pandas
2. Checked dataset structure using info() and describe()
3. Identified missing values
4. Visualized distributions using histograms
5. Detected outliers using boxplots
6. Analyzed relationships using correlation matrix and pairplots
7. Identified trends using bar plots

## 📊 Key Insights

* Female passengers had higher survival rates than males
* Passengers in higher class had better survival chances
* Fare column contains outliers
* Age distribution is slightly skewed
