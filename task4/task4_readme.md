# Logistic Regression - Binary Classification

## Objective

To build a binary classifier using Logistic Regression.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

##  Steps Performed

1. Loaded dataset (Breast Cancer dataset)
2. Split data into training and testing sets
3. Standardized features using StandardScaler
4. Trained Logistic Regression model
5. Made predictions and probability estimates
6. Evaluated using confusion matrix, precision, recall, ROC-AUC
7. Plotted ROC curve
8. Tuned classification threshold

## Key Insights

* Model performs well in distinguishing classes
* ROC-AUC score indicates strong performance
* Threshold tuning helps balance precision and recall
* Feature scaling improves model performance
