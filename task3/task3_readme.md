# Linear Regression - House Price Prediction

## Objective

To implement and understand simple and multiple linear regression models.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Steps Performed

1. Loaded dataset using Pandas
2. Selected input features and target variable
3. Split data into training and testing sets
4. Trained Linear Regression model
5. Predicted values on test data
6. Evaluated model using MAE, MSE, and R²
7. Visualized regression line (simple regression)
8. Interpreted model coefficients

## Key Insights

* Area is the most important factor affecting house price
* Model shows a positive linear relationship between features and target
* R² score indicates how well the model fits the data
* Errors (MAE, MSE) show prediction accuracy
